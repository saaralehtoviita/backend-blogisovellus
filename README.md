## Web application for writing blog posts
This application is the result of my final project on my backend programming course. The application was desiged for real needs;
in the future it will be used by an entrepeneur (physiotherapist) for writing posts on their web page. 

### Description and user roles
The main users of the application are the blog post writer, the blog post reader and the admin. 

#### Blog post writer can:
* Login
* See a list of their own posts
* Write and update their own posts

#### Blog post reader can:
* See a list of all posts ont the first page
* Search for posts by keywords
* Read all posts

#### Admin can:
* Login
* Update and delete posts

### Running the application
The application is not deployed yet. To run the application, the project needs to be downloaded and run locally. 

### Technologies 
The application has been programmed with Java and the framework used is Spring Boot. The front end part is done with Thymeleaf and Bootstrap. This version of the application is using Postgre as a database, but the database can be changed to the runtime H2-database with activating the correct profile from the application properties files.
